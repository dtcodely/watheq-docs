# المزايا الأساسية لنظام واثق

يوفر النظام حزمة من أهم المزايا التي تجعله بديلاً متكاملاً لإدارة الوثائق:

1. **البحث المتقدم الذكي (Advanced Search)**: بحث دقيق داخل خصائص الوثائق وحتى داخل نصوص المرفقات باستخدام خوارزميات PostgreSQL GIN Indexes. 
2. **استخراج النصوص (OCR)**: تحويل المستندات المصورة والمرفقات إلى نصوص قابلة للبحث مباشرة.
3. **دورة حياة الوثيقة (Disposal & Retention)**: أتمتة عمليات حذف أو أرشفة الوثائق التي تنتهي صلاحيتها وتوليد شهادات إتلاف رسمية.
4. **التجميد القانوني (Legal Hold)**: إيقاف عملية إتلاف الوثائق المطلوبة للتحقيق القانوني.
5. **تعدد اللغات (i18n)**: واجهة مستخدم (عربي / إنجليزي) تدعم الاتجاهين RTL/LTR وتعديل الثيم المظلم/الفاتح.
6. **التدقيق المعمق (Audit Trail)**: بناء سجل زمني (Timeline) لكل الإجراءات سواءً تعديل، طباعة، تحميل، أو إتلاف.
