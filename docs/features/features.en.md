# Watheq Core Features

The system offers a complete suite of features transforming it into a robust Document Management alternative:

1. **Smart Advanced Search**: Precise search within document metadata as well as directly inside attachment texts using PostgreSQL GIN Indexes.
2. **Optical Character Recognition (OCR)**: Automatically converting scanned documents and attachments into searchable text capabilities.
3. **Document Lifecycle (Disposal & Retention)**: Automate deletion or archiving of expiring documents, alongside generating formal disposal certificates.
4. **Legal Hold**: Explicitly suspending the disposal routines for documents needed in legal investigations.
5. **Multi-language UI (i18n)**: Fully responsive AR/EN interfaces with flawless RTL/LTR bidirectional support and Dark/Light theme toggle.
6. **Extensive Audit Trail**: Establishing a comprehensive timeline for every action taken, be it an edit, print, download, or disposal.
