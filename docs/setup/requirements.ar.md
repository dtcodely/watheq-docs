# متطلبات النظام

لكي يعمل نظام **واثق (Watheq)** بكفاءة، يجب استيفاء المتطلبات البرمجية التالية:

## المتطلبات الأساسية
- **Node.js**: الإصدار 18 أو أعلى.
- **Docker**: محرك Docker مثبت لتسهيل بناء وتشغيل قواعد البيانات (PostgreSQL).
- **Docker Compose**: لتشغيل الخدمات المترابطة معاً كحاوية واحدة.
- **Python**: (اختياري) لتوليد صفحات التوثيق هذه عبر MkDocs.

## قاعدة البيانات
- **PostgreSQL**: تعتبر المحرك الأساسي لقاعدة البيانات وتدعم البحث المتقدم (GIN Indexes). وينصح باستخدام الإصدار 15+.
