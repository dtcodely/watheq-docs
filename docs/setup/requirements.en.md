# System Requirements

For the **Watheq** system to run efficiently, the following technical requirements must be met:

## Prerequisites
- **Node.js**: Version 18 or higher.
- **Docker**: Docker engine installed to facilitate running databases (PostgreSQL).
- **Docker Compose**: To run related services together as a single stack.
- **Python**: (Optional) to generate these documentation pages via MkDocs.

## Database
- **PostgreSQL**: Serves as the primary database engine and supports advanced search (GIN Indexes). Version 15+ is recommended.
